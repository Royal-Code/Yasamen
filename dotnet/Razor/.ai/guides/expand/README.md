# Guides de Expansão

Este grupo reúne guides para expandir a biblioteca Yasamen.

Escopo típico:

- criar novos componentes;
- criar novos pacotes da biblioteca;
- definir CSS público e contrato visual;
- decidir composição, dependências e padrões internos;
- alinhar serviços, outlets, forms, navegação e infraestrutura;
- definir showcases e documentação da própria biblioteca.
