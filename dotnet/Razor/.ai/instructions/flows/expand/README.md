# Flows de Expansão

Este grupo concentra fluxos para expandir a biblioteca.

Fluxos ativos:

- selecionar a próxima entrega;
- planejar spec;
- criar spec;
- refinar spec;
- implementar spec;
- scaffoldar pacote da biblioteca quando a spec pedir isso.

Regra:

- estes arquivos são o caminho canônico para o fluxo de expansão da biblioteca.
