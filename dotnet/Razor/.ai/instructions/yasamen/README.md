## Escopo esperado

- instalar e configurar a biblioteca;
- criar projetos front-end que usam Yasamen;
- criar, editar e evoluir telas;
- configurar serviços, temas e showcases em apps consumidores.

## Regra de escalonamento

- usar `app-spec` quando o problema estiver no nível do app;
- usar `screen-spec` quando o problema estiver no nível da tela;
- usar `lib-spec` quando surgir expansão da biblioteca;
- usar estas instruções quando o trabalho já estiver claro e puder seguir direto.
