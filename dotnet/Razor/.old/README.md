# Arquivos Arquivados

Esta pasta guarda arquivos antigos, legados ou fora do fluxo ativo.

## Regra de Uso

- `.old/` não é entrada ativa para IA nem para o Copilot.
- arquivos aqui servem como histórico, referência ou transição.
- antes de reutilizar algo daqui, a regra é revisar e reintroduzir conscientemente no local canônico.
