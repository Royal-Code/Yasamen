# Artefatos do Codex

Este diretório concentra arquivos específicos do Codex dentro do repositório.

Regra de organização:

- `.codex/` guarda skills, instruções e metadados específicos do Codex;
- `.ai/` continua a guardar conteúdo agnóstico de ferramenta, como catálogos, guias, templates e fluxos de domínio.
